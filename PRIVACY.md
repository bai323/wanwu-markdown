# 隐私说明

万物 Markdown 采用本地优先的工作方式。默认情况下，采集结果、Markdown、HTML 报告、结构数据和 Obsidian 输出都保存在你的电脑上。

## 会读取什么

- 你主动输入的网页或对话 URL。
- 你主动导入的 JSON 文件。
- 你选择的本机 Chrome 配置和 Sider 插件会话数据。
- 你选择的 Obsidian Vault 路径。

## 不会主动做什么

- 不会在后台扫描所有浏览器历史记录。
- 不会自动上传你的对话或笔记内容。
- 不会绕过平台登录、权限或付费限制。

## 使用提醒

请只处理你有权保存、整理和继续使用的内容。多人聊天、客户资料、私密对话和未公开文档在导出前应先获得授权，并在分享或训练前做脱敏处理。
